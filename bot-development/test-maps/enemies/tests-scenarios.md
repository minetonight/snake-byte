# Epic 4 Enemy/Adversarial Scenarios

These maps target Epic 4 behaviors:
- Reachability envelope masking (danger map)
- Localized alpha-beta in close enemy overlap
- Dynamic role reassignment for multi-snake cooperation

Run with:
- `cd bot-development/simulation/`
- `python3 run_simulation.py "<bot1>" "<bot2>" --map "<map-file>"`

Primary benchmark opponent for these tests is `Boss.py` (WAIT bot).
